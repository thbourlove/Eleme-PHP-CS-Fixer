Eleme-PHP-CS-Fixer
==================

Fix php code.

### How to install

    wget https://raw.github.com/thbourlove/Eleme-PHP-CS-Fixer/master/install.sh -O - | sh

### How to use it

default mapping is `<leader>ff` (`,ff`)
